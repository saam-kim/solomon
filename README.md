# 편의점 솔로몬: 선을 긋는 연습

고등학교 2학년 법교육용 선택형 비주얼 노벨 게임입니다.

## 실행

```bash
npm install
npm run dev
```

브라우저에서 `http://127.0.0.1:5173`을 열면 됩니다.

## 포함 기능

- 제공 PNG 일러스트 기반 배경, 캐릭터, 소품 레이어
- 11개 본편 장면과 엔딩 분기
- 선택 플래그와 상태값 기반 후속 장면/엔딩 반영
- 법 포인트 정리 카드
- localStorage 저장/불러오기
- 글자 크기, 빠른 진행, 다시보기 로그
- 교사용 장면 바로가기, 선택 기록, 법 개념 정리, 토론 질문
