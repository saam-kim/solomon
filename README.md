# 편의점 솔로몬: 선을 긋는 연습

고등학생이 편의점 알바생 지후의 입장에서 노동법, 민법, 거래 질서와 안전 책임을 경험하는 선택형 비주얼노벨입니다. 법적 기준뿐 아니라 그 기준을 사람에게 어떻게 설명하는지도 선택과 엔딩에 반영됩니다.

## 실행 방법

```bash
npm install
npm run dev
```

PowerShell 실행 정책으로 `npm.ps1` 실행이 제한된 환경에서는 다음 명령을 사용합니다.

```powershell
npm.cmd install
npm.cmd run dev
```

프로덕션 빌드와 코드 검사는 다음과 같습니다.

```bash
npm run lint
npm run build
```

## 프로젝트 구조

- `src/data`: 대본, 선택지, 법 포인트, 엔딩, 후반 회수와 교사용 데이터
- `src/components`: 비주얼노벨 화면, 대화·선택·엔딩·교사용 UI
- `src/hooks`: 게임 상태와 장면 진행
- `src/utils`: 에셋 매핑, 조건·점수·엔딩 판정, 결과 보고서, 저장/불러오기
- `src/types`: 게임 데이터 타입
- `src/assets/illustrations`: PNG 배경, CG, 캐릭터, 소품과 UI 이미지

이미지는 `import.meta.glob`으로 중첩 폴더까지 자동 탐색합니다. 참조한 파일이 없으면 사람이나 소품을 임시 도형으로 그리지 않고 텍스트 placeholder를 표시합니다.

## 기본 조작

- 클릭 / `Space`: 다음 대사
- `Enter`: 다음 대사 또는 첫 번째 선택지
- 숫자키 `1`~`5`: 해당 선택지
- `Esc`: 열린 로그, 상태, 설정 또는 교사용 패널 닫기
- `SETTINGS`: 글자 크기, 빠른 진행, 음소거 UI
- `AUTO`: 대사를 자동으로 진행하고 선택지에서 정지
- `TEACHER`: 장면 바로가기, 선택 기록, 법 개념, 토론 질문과 결과 복사

진행 상태는 브라우저 localStorage의 `convenience-solomon-save-v1` 키에 저장됩니다. 외부 서버와 Firebase는 사용하지 않습니다.

## 수업 활용

1. 학생 개별 플레이
2. 엔딩 결과와 선택 분석 확인
3. 교사용 패널에서 학생의 선택 기록과 내부 상태 확인
4. 결과를 텍스트로 복사해 선택 경향 공유
5. 장면별 또는 엔딩별 토론 질문으로 후속 활동 진행

교사는 현재 상태를 유지하거나 초기화한 뒤 원하는 장면으로 바로 이동할 수 있습니다. 빔프로젝터 시연에서는 설정 패널의 큰 글자와 빠른 진행 기능을 활용할 수 있습니다.
